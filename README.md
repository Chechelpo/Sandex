CLI tool for extracting files in a sandbox, only available for Linux.
<p>
Dependencies:
    -[bwrap](https://github.com/containers/bubblewrap): for sandbox
    -[bsdtar](https://github.com/libarchive/libarchive): for zip and tar extraction
    -[unrar](https://github.com/aawc/unrar): for rar extraction
</p>
<p>
For details on how the sandbox is configured, see sandbox.py. Future versions will support adding sandboxing overrides and creating profiles
</p>
